# 第1幕 記憶の欠片
## エピソード2「謎めいた少女」

午後の陽が傾き始めた中庭に、片桐悠はひとり座っていた。錆びたベンチの背に寄りかかりながら、彼はしばし眼を閉じていた。微かな風が頬を撫で、制服の襟を揺らす。それでも消えないのは、頭の奥に静かに根を張るような鈍い痛みだった。

近頃、眠るたびに見る夢があった。白い光の中で名前を呼ぶ声。振り返ろうとしても身体は動かず、声の主に手を伸ばそうとすると、視界が闇に引き裂かれて——そして目覚める。

いまの自分は、自分でいて自分ではない。何かが抜け落ちている。何か、重要なものが——。

ベンチの背後の影に気付いたとき、悠は眉をひそめた。気配はしていたはずなのに、声も足音もなかった。

「君、思い出せないんでしょう？」

その声は妙に澄んでいた。日常の喧騒に紛れない、そこだけ切り取られたような音。悠は目を開け、振り返る。

そこに立っていたのは、見覚えのない少女だった。

涼しげな黒のワンピースに、風になびく銀の髪。均整の取れた顔立ちは冷たさと儚さを併せ持ち、片眉をわずかに上げるその表情は、大人びていて挑発的だった。

「……誰だ？」

「質問に質問を重ねるの？　野暮だね」

彼女は少し唇を歪めるようにして笑った。その笑顔には温度がなく、どこか演技じみてすらいた。

「でも、まあ良いさ。七瀬凛って名乗ればいいかな。今のあなたにとっては、それくらいで十分」

「名乗ればいいって……つまり、本名じゃないってことか？」

「さあ、どうかな。私自身、自分という曖昧な存在を名乗るたびに、不思議な気分になるよ。まるで物語の登場人物でも語ってるみたいで」

そして、彼女は悠の隣に座った。距離はぎりぎりで、人と人との境界線を越えるほどの近さだった。風が掻き回す彼女の髪が、悠の頬をかすめる。

悠は思わず息をのむ。

その瞬間、漠然とした記憶の底から浮かび上がる感覚があった。髪の香り、声の抑揚、目の奥の淋しさ——知っている。確かに知っていたはずなのに、言葉では再現できない。まるで夢の終わりで、指の隙間からこぼれる水滴のように、感覚は逃げていく。

「俺の記憶について――何を知ってる？」

「たくさん。だけどその中には、あなた自身が閉ざした扉の向こうにあるものもある」

彼女は悠の胸元に視線を落とした。

「記憶って不思議。脳にあるようでいて、心にしまい込まれているの。都合が悪いことは見ないふりをする。怖いものほど手放す。でも、その代償はいつだって大きい。とくに今回みたいな、感情ごと切り離されたような喪失はね」

「感情ごと……」

「あなたには、忘れたくなる理由があった。でもその結果、大切な何かも一緒に殺した」

「殺した？」

「感覚的な比喩よ。もっとも、ある意味では真実でもあるけれど」

七瀬凛はそっと手を伸ばすと、悠の制服の内ポケットに何かを滑り込ませた。

その際に指先が一瞬だけ触れ合った。冷たいようで温かく、しかし彼女の表情には何らの感情も浮かばない。

「手品みたいにね、ポケットには何が入ってると思う？」

悠は無言で手を差し入れた。小さな硬質の物が指に触れる。取り出してみると、それは薄い金属製の不思議な形の鍵だった。古びているが、その造形には妙な意匠が施されており、誰かの意図が込められていることだけは伝わってきた。

「……なんだ、これ」

悠は問いかけた。しかし凛は答えず、わざと無関心を装ったように立ち上がる。

「答えは、次の場所にある。鍵は扉を開くためにあるんでしょう？」

「だから何なんだよ、お前は。どうして俺の記憶のことをそんなに……」

彼女は一瞬、立ち止まり、振り返らないまま呟いた。

「だって私は……ずっと、あなたに思い出してほしかったから。それだけのために、ここに来たのかもしれない」

風が吹く。夕陽に照らされた中庭で、木々がざわめいた。空気が焼けるような熱さから、どこか雨の匂いすら感じる湿った冷たさに変わる。

視線を戻したとき、凛の姿はなかった。まるで景色と同化していたかのように、唐突に消えていた。

悠は鍵を手の中で握りしめた。彼女が残した言葉一つ一つが、どこかに閉じ込められていた記憶をかき混ぜる。

「案内人……」

その声もまた耳に残っていた。見覚えのない少女。だが、雰囲気、何気ない仕草、匂いまでも——懐かしさを含んでいた。

悠はベンチから立ち上がり、夜の近づく空を仰いだ。その胸には確かな予感があった。

この鍵が開ける扉の先に、自分の世界を変えてしまう真実がある。失われた、真鍋沙耶との日々。あの日、崩れた記憶。

彼はポケットに鍵をしまい、そっと自分の胸に手を当てた。弱く、しかし確かに脈打つものが、ここに生きている記憶を告げている。

それは甘酸っぱく、けれど微かに苦い。初恋と、裏切りと、喪失と——青春の中に埋もれた記憶の味だった。

「……思い出してやるさ。俺の選択で、全部」

悠の足は、知らず旧新聞部のある校舎方面へと向かっていた。

青く冷え始めた風が、その背をゆっくりと押した。
