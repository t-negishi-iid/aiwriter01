# 現代的なSF・ファンタジーの作品世界と舞台設定

## デジタルディストピア設定
### 基本的な世界観
- 高度に発達したテクノロジーが支配する近未来社会
- AIや監視システムによる管理社会
- 現実と仮想の境界が曖昧な世界
- デジタル格差による階層社会

### 特徴的な要素
- 巨大テクノロジー企業による支配
- データやプライバシーを巡る争い
- サイバースペースでの戦い
- アナログ回帰を求める反体制勢力
